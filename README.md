# Модель прогнозирования успеха мероприятия

Этот репозиторий содержит код и ресурсы для магистерской диссертации под названием **"Исследование отраслевых проблем ивент-индустрии и каталогизация цифровых решений с применением искусственного интеллекта"**. Проект направлен на разработку предиктивного инструмента для оценки и прогнозирования успешности мероприятий с использованием математического моделирования и машинного обучения.
## Обзор проекта

Инструмент предназначен для:
- Анализа ключевых параметров мероприятий (общая длительность мероприятия, размер аудитории, длительность награждения, количество активностей, длительность выступлений, длительность речи ведущих).
- Прогнозирования оценки успешности мероприятия по шкале от 0 до 10.
- Предоставление информации о том, как различные факторы влияют на успешность мероприятия.
- Применимости в образовательных и профессиональных условиях управления мероприятиями.

## Основные характеристики

### Математическая модель
В основе инструмента лежит математическая модель, которая рассчитывает оценку успешности на основе:
1. **Взвешенные параметры**: Каждому фактору присваивается определенный вес.
2. **Нелинейные поправки**: Взаимодействие между параметрами моделируется с помощью матрицы.
3. **Регулировка энтропии**: Обеспечивает сбалансированную структуру событий, штрафуя несбалансированные распределения параметров.

### Модель машинного обучения
Для улучшения прогнозов используется регрессор градиентного усиления:
- **Предварительная обработка данных**: включает стандартизацию и генерацию полиномиальных признаков.
- **Настройка гиперпараметров**: оптимизировано с помощью GridSearchCV для лучшей производительности.
- **Показатели производительности**: достигает MAE (средняя абсолютная ошибка) 0,2 и R2 (коэффициент детерминации) 0,97.

## Как это работает

### Входные данные
Инструменту требуются следующие параметры для каждого события:
- `duration`: Общая длительность мероприятия (минуты).
- `audience_size`: Размер аудитории.
- `award_duration`: Длительность награждения (минуты).
- `num_items`: Количество активностей.
- `perf_duration`: Продолжительность выступлений (минуты).
- `host_speech_duration`: Продолжительность речи ведущих (минуты).

### Вывод
- **Оценка успеха**: значение от 0 до 10, указывающее на прогнозируемый успех события.
- **Анализ**: визуализации (гистограммы, корреляционные графики) для интерпретации результатов.

## Структура репозитория
- `Расчет оценки успешности и анализ результатов.ipynb`: Содержит реализацию математической модели.
- `Модель прогнозирования успешности мероприятия.ipynb`: Содержит реализацию модели машинного обучения.
- `events_hall_dataset.csv`: Образец набора данных, использованный для обучения и тестирования.
- `events_hall_dataset_with_scores.csv`: Набор данных дополнен показателями успешности, предсказанными моделью.

## Использование

### Требования
- Python 3.9+
- Зависимости: `numpy`, `pandas`, `scikit-learn`, `matplotlib`

### Пошаговое руководство
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Berusov/Event_predict.git
   cd Event_predict
   ```

2. Установить зависимости:
   ```bash
   pip install numpy pandas scikit-learn matplotlib
   ```

3. Откройте и запустите `Расчет оценки успешности и анализ результатов.ipynb` в Jupyter Notebook или Google Colab для расчета оценки. 
Откройте и запустите `Модель прогнозирования успешности мероприятия.ipynb` в Jupyter Notebook или Google Colab для прогнозирования оценки.

4. Чтобы предсказать успешность пользовательского события, заполните соответствующие поля формы ввода и нажмите кнопку "Рассчитать оценку".

## Результаты
- Модель была протестирована на данных Творческого центра Донского государственного технического университета, достигнув высокой точности в прогнозировании успеха мероприятия.
- Модель помогает выявить слабые места в планировании мероприятия и повысить объективность оценки его успешности.
